## 完整时间线

| 年份 | 事件 | 意义 |
|------|------|------|
| **1958** | 出生于北京 | 成长于文革前后的社会环境，亲历社会动荡，为日后关注人性与犯罪埋下底色 |
| **1970s** | 青少年时期，经历文化大革命尾声 | 特殊历史时期的成长经历，影响其对"环境塑造人"的深层认知 |
| **1978** | 考入中国人民大学，就读哲学系（本科） | 哲学训练奠定其思维方式：追问"为什么"而非"是什么"，后来成为其犯罪心理分析的底层逻辑 |
| **1982** | 本科毕业，进入中国人民公安大学任教 | 职业起点，开始接触犯罪学领域；公安大学的实践导向影响其研究风格 |
| **1980s 中期** | 开始系统研究犯罪心理学，参与案件分析工作 | 从理论走向实践，积累大量真实犯罪案例，为后来的"早年经历→犯罪行为"理论提供素材 |
| **1990s** | 在中国人民公安大学犯罪心理学教研室深耕，逐步建立研究体系 | 学术积累期；开始注意到犯罪者早年家庭经历的规律性，思想转折的酝酿期 |
| **2004** | **马加爵案**（2004年2月）发生后，李玫瑾接受媒体采访，对案件进行犯罪心理分析 | **走向公众视野的关键节点**。她的分析聚焦马加爵的童年孤独感与人际敏感性，首次在全国性媒体上展示"从早年经历解读犯罪"的分析框架 |
| **2004-2006** | 频繁参与央视《心理访谈》等节目，开始系统向公众讲解犯罪心理学 | 从学术圈走向大众传播；电视媒体成为其思想传播的主要渠道 |
| **2008** | 出版《犯罪心理研究——在犯罪防控中解读犯罪心理》 | 学术代表作之一，系统呈现其犯罪心理学研究体系 |
| **2010** | **药家鑫案**（2010年10月）发生，李玫瑾参与案件心理分析 | 再次引发全国关注；她对"高压教育→情绪管理缺失"的分析引发广泛讨论，也首次遭遇较大规模的公众质疑（被批评为"为杀人犯辩护"） |
| **2011** | 出版《幽微的人性》（上海三联书店，2014年再版） | **重要著作**。将犯罪心理学研究通俗化，面向大众读者；书中大量案例分析奠定其公众知识分子形象 |
| **2011-2013** | 思想重心开始从"犯罪分析"向"犯罪预防→儿童早期教育"转移 | **思想转折点 ★**。核心逻辑：既然犯罪根源在早年，预防犯罪的最有效手段是改善早年养育。研究重心从"事后分析"转向"事前预防" |
| **2013** | 开始在各地高校、社区、企业大规模开展育儿主题讲座 | 从犯罪心理学家转型为"家庭教育专家"的公众形象确立期；讲座内容开始系统化 |
| **2015** | 参与央视《开讲啦》节目，面向年轻一代讲述心理抚养理念 | 传播渠道扩展至年轻受众；"心理抚养"概念开始广泛传播 |
| **2016** | 正式提出并系统阐述"心理抚养"理论框架 | **核心理论成型 ★**。将"0-6岁情感陪伴"、"依恋关系"、"延迟满足训练"整合为完整理论体系，与"物质抚养"形成对照概念 |
| **2017-2018** | 讲座视频在B站、微博等平台大规模传播，成为现象级内容 | **互联网传播爆发期**。从电视媒体时代进入社交媒体时代，受众从中老年父母扩展至年轻父母群体 |
| **2017** | 主译出版《犯罪心理学》第11版（[美]柯特·R.巴托尔、安妮·M.巴托尔著，ISBN: 9787518412655） | 学术翻译工作，将国际犯罪心理学权威教材引入中国；体现其在专业领域的持续学术贡献 |
| **2019** | 从中国人民公安大学退休（教授职位） | 退休后反而获得更多公开演讲和写作时间；学术身份转为荣誉/兼职，公众活动不减反增 |
| **2021** | 出版《心理抚养》 | **最重要著作 ★**。将多年讲座内容系统整理成书，成为畅销书；正式确立"心理抚养"作为其核心理论标签。出版后引发广泛讨论，登上各大图书榜单 |
| **2021-2022** | 《心理抚养》引发的"妈妈应亲自带孩子"争议持续发酵 | 女权视角批评者指出其论点将育儿责任单方面压在母亲身上；李玫瑾多次公开回应，坚持立场但补充说明"父亲同样重要" |
| **2022** | 持续参与公开讲座，开始更多关注青少年心理健康议题（抑郁、焦虑、手机成瘾） | 回应社会热点：青少年心理健康问题在疫情期间显著上升，她的分析框架扩展至新议题 |
| **2023** | 多次接受权威媒体采访，就"躺平"、"内卷"等社会现象从心理抚养角度发表观点 | 理论应用扩展：将早年养育理论与当代青年社会现象连接，保持公众话题的相关性 |
| **2023** | 参与讨论"未成年人犯罪低龄化"立法议题，就《预防未成年人犯罪法》修订发表专业意见 | 回归犯罪心理学专业身份，在政策层面发挥影响力 |
| **2024** | 继续活跃于公开讲座和媒体，就"游戏成瘾"、"青少年抑郁"等议题发声 | 持续更新理论应用场景；关注数字时代对儿童心理发展的影响 |
| **2024** | 相关讲座视频在B站累计播放量持续增长，部分视频播放量超千万 | 互联网影响力持续扩大，成为中国家庭教育领域最具影响力的公众知识分子之一 |
| **2025** | 出席多场教育论坛，就"AI时代的儿童教育"发表观点；强调情感连接在技术时代的不可替代性 | 理论与时代议题结合：将"心理抚养"框架延伸至AI、数字化背景下的育儿挑战 |
| **2025** | 持续通过微博（@李玫瑾）发布育儿观点，与网友互动 | 社交媒体运营稳定，保持与年轻父母群体的直接对话 |
| **2025-2026** | 关注"生育率下降"社会背景下的家庭教育议题，就"独生子女养育"、"高龄父母育儿"等新型家庭结构发声 | 理论持续与社会现实对话；在人口结构变化背景下重新阐释"心理抚养"的紧迫性 |
